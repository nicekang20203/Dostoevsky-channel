# Dostoevsky — The Soul of Russian Literature

A companion website for the YouTube channel **역사를 품은 문학 / Literature Embracing History**.

## Overview

An immersive, bilingual (EN/KO) website exploring Fyodor Dostoevsky's life, family, works, and legacy. Features a dark academia aesthetic with 19th-century literary journal influences.

## Sections

1. **Family & Origins** — Parents, siblings, and childhood at Mariinsky Hospital
2. **Life & Times** — Interactive timeline from birth to death
3. **Anna Snitkina** — The woman who saved Dostoevsky (stenographer, manager, wife, legacy guardian)
4. **Major Works** — Detailed descriptions of six novels with themes
5. **Video Episodes** — Embedded YouTube playlists
6. **Quotes** — Carousel of famous passages

## Files

```
index.html   — Main page structure
style.css    — Dark academia styling with CSS variables
script.js    — Navigation, language toggle, animations, quote carousel
anna.html    — Standalone interactive page on Anna (optional)
```

## Features

- **Bilingual**: EN/KO toggle with localStorage persistence
- **Responsive**: Mobile-first with breakpoints at 480px, 768px, 1024px
- **Animated**: Scroll-triggered reveals, timeline animations, quote carousel with touch support
- **Accessible**: Keyboard navigation, ARIA labels, semantic HTML
